# 🐺 IRON WOLF - Martial Arts Mastery

هذا هو مشروعك الخاص بتدريبات الفنون القتالية المدعوم بالذكاء الاصطناعي. إليك دليلك لفهم الكود وكيفية تشغيله.

## 📁 أين يوجد الكود؟ (هيكل المشروع)

- **`src/app/`**: هنا توجد "واجهات" التطبيق (Pages).
  - `page.tsx`: لوحة التحكم الرئيسية (Dashboard).
  - `generator/page.tsx`: صفحة توليد التمارين بالذكاء الاصطناعي.
  - `history/page.tsx`: سجل التدريبات.
  - `goals/page.tsx`: صفحة الأهداف والمهمات.
- **`src/ai/flows/`**: هنا يوجد "عقل" التطبيق. ملف `generate-personalized-workout.ts` هو المسؤول عن التواصل مع Gemini AI لإنشاء الجداول التدريبية.
- **`src/components/`**: تحتوي على العناصر المتكررة مثل الأزرار (Button)، القوائم (Navigation)، والبطاقات (Card).
- **`public/`**: للصور والأيقونات التي تريد استخدامها.

## 🚀 كيف تشغل البرنامج الآن؟

1. افتح الـ **Terminal** (نافذة الأوامر في الأسفل).
2. اكتب الأمر التالي واضغط Enter:
   ```bash
   npm run dev
   ```
3. سيظهر لك رابط (غالباً `http://localhost:9002`)؛ اضغط عليه ليفتح التطبيق في المتصفح أمامك.

## 🌐 كيف تجعل البرنامج "تطبيق حقيقي" يفتحه الناس؟ (النشر)

بما أن المشروع مبني بتقنيات حديثة (Next.js & Firebase)، فإن نشره يتم عبر **Firebase App Hosting**:

1. **ارفع الكود على GitHub:** قم بإنشاء مستودع (Repository) جديد وارفع ملفاتك عليه.
2. **اربطه بـ Firebase:**
   - اذهب إلى [Firebase Console](https://console.firebase.google.com/).
   - اختر مشروك، ثم اذهب إلى **App Hosting**.
   - اضغط على **Get Started** واربط حساب GitHub الخاص بك.
3. **النشر التلقائي:** بمجرد الربط، في كل مرة تضغط فيها على "Save" أو تقوم برفع تحديث، سيقوم Firebase ببناء "البرنامج" ونشره فوراً على رابط (URL) يمكنك إرساله لأصدقائك.

## 🛠 التقنيات المستخدمة
- **Next.js 15**: الإطار البرمجي الأساسي.
- **Genkit**: لتشغيل ذكاء Gemini AI.
- **Tailwind CSS**: لتصميم الواجهات بشكل احترافي.
- **Lucide Icons**: للأيقونات.
